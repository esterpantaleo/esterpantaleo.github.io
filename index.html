tools

  1. etytree

    Website · Github · Bitbucket · grant · demo
    An interactive tool to visualize etymologies and find etymologically connected words + an RDF database of etymological connections extracted from the English Wiktionary. Developed with a Wikimedia Foundation individual engagement grant.

    TECHNICAL CONTENT: d3.js · RxJS · linked open data · regex · RDF databases · SPARQL · grant writing · java

visualizations

  1. ANPR dashboard

    Website
    A real time interactive dashboard showing progress in the construction of a National Resident Population Register. Developed for the Digital Transformation Team of the Italian Government.

    TECHNICAL CONTENT: d3.js · mapbox · webpack · open government data


  2. Star cluster evolution

    Website · Github
    An animated and interactive version of the HR diagram that evolves in time from birth to an age of 5 billion years using simulated data of a cluster of 100-star similar to our Sun's birthplace and images from the World Wide Telescope. Made for the Adler Planetarium, Chicago.

    TECHNICAL CONTENT: d3.js · bash · data cleaning


  3. Flusso regionale studenti di Ateneo

    Website · Github

    An interactive visualization of the inbound, outbound, and stationary flow of Italian students per region, area of study, and sex using real data from MIUR. Made for the Digital Team of the Italian Government.

    TECHNICAL CONTENT: d3.js · data cleaning · open government data


  4. Mappa dei quartieri

    Website · Github
    An interactive dashboard to explore and compare different neighborhoods in major Italian cities using data from ISTAT and other public sources. Currently under development for the Digital Team of the Italian Government.

    TECHNICAL CONTENT: React + d3.js · mapbox · webpack · open government data


  5. Infografica UNIBO

    Website · Github
    A visualization using open data on schools and courses of the University of Bologna. Currently under development for the Digital Team of the Italian Government.

    TECHNICAL CONTENT: d3.js · open data


research

  1. altra

    Paper · Supplementary material · Github
    A Bayesian method for simultaneous transcript reconstruction and abundance estimation with RNA-Seq data from multiple samples. Developed at the University of Chicago.

    TECHNICAL CONTENT: genomics · Bayesian statistics · data modeling · C++ · R · bash


  2. multiseq

    Github
    Development of an R package for multiscale analysis, ongoing work in the Stephens lab at the University of Chicago. The embedded model considers data as an inhomogeneous Poisson process and tests for differences in underlying intensities using a Bayesian multi-scale model.

    TECHNICAL CONTENT: genomics · Bayesian statistics · data modeling · R


  3. monomvn

    Paper
    Contributed to "Shrinkage Regression for Multivariate Inference with Missing Data, and an Application to Portfolio Balancing" published on Bayesian Anal. Vol. 5, N. 2 (2010), 237-262, by extending the model to data with fat tails. Developed at the University of Cambridge.

    TECHNICAL CONTENT: econometrics · Bayesian statistics · data imputation · C++


  4. Covariance estimation

    Paper · Github
    Comparison of different portfolio optimization techniques "When do improved covariance matrix estimators enhance portfolio optimization? An empirical comparative study of nine estimators" published on Journal of Quantitative Finance, Vol. 11 (2011) - Issue 7.

    TECHNICAL CONTENT: econometrics · shrinkage · R


  5. Hausdorff clustering

    Paper 1 · Paper 2
    Development of a new clustering algorithm based on the Hausdorff distance, also applied to financial time series.

    TECHNICAL CONTENT: econometrics · clustering · matlab
